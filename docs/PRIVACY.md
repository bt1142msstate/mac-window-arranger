# Privacy Policy

Mac Window Arranger does not collect, transmit, sell, or share personal data.

The app reads the list of running applications and visible windows on the Mac so
you can choose which windows to resize or arrange. Window titles and saved
layouts stay on the device. Saved layouts are stored locally in the app's
preferences using `UserDefaults`.

Mac Window Arranger requests Accessibility permission so it can move and resize
windows in other apps. The app does not use analytics, advertising, tracking, or
network services.
